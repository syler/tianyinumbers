log4cxx 使用方法

14 July 2011

下载链接:

apr log4cxx

第一步安装apr-1.4.5，顺序不能错，它必须首选安装

$tar -zxvf apr-1.4.5.tar.gz
$cd apr-1.4.5
$./configure --prefix=/usr/local/apr
$make
$su root
$make install

然后安装apr-util-1.3.12

$tar -zxvf apr-util-1.3.12.tar.gz
$cd apr-util-1.3.12
$./configure --prefix=/usr/local/apr-util --with-apr=/usr/local/apr
$make
$su root
$make install

configure选项 –with-apr=/usr/local 指定apr库的位置 #### 安装log4cxx了

在这之前,必须要改三个文件:

$vim src/examples/cpp/console.cpp 在这个文件里添加#include <stdio.h>和#include <string.h>
$vim src/main/cpp/socketoutputstream.cpp 在这个文件里添加#include <string.h>
$vim src/main/cpp/inputstreamreader.cpp 在这个文件里添加#include <string.h>

$tar -zxvf apache-log4cxx-0.10.0.tar.gz
$cd apache-log4cxx-0.10.0
$./configure --prefix=/usr/local/log4cxx --with-apr=/usr/local/apr --with-apr-util=/usr/local/apr-util

$sudo make
$su root
$make install

最后,在terminal执行:

sudo ldconfig

如果 在执行的时候，你可能会得到下面的错误：

error while loading shared libraries: liblog4cxx.so.10: cannot open shared object file: No such file or directory

查找半天发现在/usr/local/lib下有liblog4cxx.so.10文件，这个问题是因为你没有设置LD_LIBRARY_PATH环境变量造成的。

执行

export LD_LIBRARY_PATH=/usr/local/lib
