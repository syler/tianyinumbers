解決Prism中嵌套的Region在RegionManager中找不到的問題

示意圖

Region A裝載ModuleA，RegionB裝載ModuleB，Region C裝載Module C 其中，Module A內部還有 Region A1和A2，現在需要Region A1的內容導航爲其他內容。

this.regionManager.RegisterViewWithRegion(RegionA1, typeof(RegionA1Content1));

頁面可以正常顯示，RegionA1Content顯示在RegionA1中，但是用RequestNavigate進行導航時失敗，經查是由於RegionManager不存在RegionWizardMainContent這個Region。 導航語句爲：

this.regionManager.RequestNavigate(RegionA1, “RegionA1Content2”);

上一句是失敗的。 解決方法是使用RegionManager.SetRegionManager：

RegionManager.SetRegionManager(this.regionA1Region, regionManager);

帮助选择电信手机号的小程序

点击Search，向电信网站索取数据，可以把获取次数调得大些，不过建议不超过1000次，因为同时发出1000个http请求，对服务器有压力，而且，次数越多重复的号码获取得越来越多，新号码就越来越少。 Visual Studio 2013下开发，WPF/C#。 开源地址：http://git.oschina.net/syler/tianyi-numbers 暂时没有提供编译好的二进制exe

Save可以将获取到的号码保存到一个文件 绿色的Drag To Load Saved是将一个保存好的文件加载到程序中。 过滤是从已获得的号码中筛选心仪的号码。
